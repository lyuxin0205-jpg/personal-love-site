# CHANGELOG

## 当前阶段：低饱和日系情侣生活记录网站

本阶段已经完成从早期模板感、黑金感、商务感，到自然光、黄绿蓝、胶片生活记录感的转向。

## 已完成内容

### 项目基础

- 创建 Next.js + Tailwind CSS + Framer Motion 项目。
- 使用 App Router，页面入口为 `app/page.tsx`。
- 配置基础 SEO metadata，内容来自 `data/site.ts`。
- 使用 `next/image` 管理图片展示。
- 使用本地音频文件作为默认音乐。
- 增加启动脚本：
  - `启动网站.cmd`
  - `开发模式-修改后自动刷新.cmd`
  - `start-site.bat`

### 内容管理

- 将核心内容集中到 `data/site.ts`。
- 增加中文注释，方便后续非开发者修改。
- 已集中管理：
  - 情侣姓名
  - 私密密码
  - 恋爱开始日期
  - 首页标题、副文案、背景图
  - 导航、按钮、placeholder、footer
  - 时间轴故事
  - 相册图片
  - 相册空位
  - 生活碎片
  - 双人日记默认内容
  - 留言板默认内容
  - 城市记录
  - 纪念日
  - 愿望清单
  - 音乐标题和路径
- 增加 `contentSource`，为未来接入 CMS 预留结构。

### 视觉调整

- 放弃黑金色调，改为低饱和黄绿蓝。
- 建立米白、奶绿、青绿、浅蓝、深青灰的视觉系统。
- 减少强玻璃拟态和商业卡片感。
- 去掉抽象小人插画感。
- 去掉大部分装饰性图标，只保留功能性图标。
- 相册改为胶片照片墙，支持轻微旋转、错位和不同照片比例。
- 增加相册空位，让页面像长期生长的私人相册。
- 增加生活碎片，让页面从“设计展示”更接近日常记录。
- 首屏加入日期、城市、天气、生活感文案。
- 全站中文化，避免英文模板文案。

### 动效与交互

- Hero 背景增加慢速呼吸和鼠标轻微视差。
- 页面进入使用淡入、轻微位移和 blur。
- 滚动出现使用 Framer Motion。
- 相册 hover 使用轻微上移和旋转归正。
- 部分照片支持 Live Photo 风格轻微动态。
- 照片点击可打开大图预览。
- 日记、留言、愿望清单使用 localStorage 本地持久化。
- 底部音乐播放器支持播放、暂停、进度、音量。

### 移动端

- 页面采用移动端优先布局。
- Hero 标题使用响应式 `clamp()`。
- 相册在移动端为单列，平板和桌面逐步多列。
- 日记、留言、纪念日、愿望清单在移动端纵向排列。
- 底部为音乐播放器和安全浏览预留空间。

### 工程与验证

- 增加 `scripts/lint.mjs` 自定义检查。
- 增加 `scripts/browser-qa.mjs` 浏览器 QA 脚本。
- QA 脚本已改为自动使用空闲端口，避免和 3000 预览冲突。
- 生产构建已通过。
